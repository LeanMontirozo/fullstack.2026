# Tuntiharjoitukset

This folder contains 4 focused Express.js examples based on the Workshop 03 tasks.

## Examples

1. example-1-task1-2.js
   - Task 1: Create the Express app instance
   - Task 2: Serve static files from public/
2. example-2-task3.js
   - Task 3: Route handlers for /, /about, /contact
3. example-3-task4.js
   - Task 4: JSON API endpoint /api/time
4. example-4-task5.js
   - Task 5: 404 and 500 error handlers

## Public Assets

The public/ folder here mirrors the files used in the starter project so the examples can serve pages and error views.

## How to Use

- Use these files as reference snippets for your own server.js in starter/.
- If you want to run an example directly, make sure Express is installed (e.g., run npm install in starter/ or add a package.json here).
- Then run from the repo root:
  node tuntiharjoitukset\example-1-task1-2.js

Note: The examples have app.listen commented out. Uncomment it to start the server.
